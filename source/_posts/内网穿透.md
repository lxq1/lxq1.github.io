# 内网穿透

## 下载解压frps
```

wget http://ip:8088/frp.tar.gz

tar -xzf frp.tar.gz

./frps -c ./frps.ini


```

## 查询版本


```
cat /etc/redhat-release 
```




## 关闭防火墙

```
#centOS
firewall-cmd --state #查询状态
systemctl stop firewalld.service #停止firewall
systemctl disable firewalld.service #禁止firewall开机启动

```
## 编辑配置文件


```

vi frps.ini 
按键i进行输入状态
改配置
按键ESC退出输入状态
按键:wq!退出保存
cat frps.ini #查看

```
